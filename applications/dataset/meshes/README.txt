This folder contains the datasets D1 (downloadable part only), D2 and D3 by Liu L, Ye C, Ni R, Fu X-M, Chai (http://staff.ustc.edu.cn/~fuxm/projects/ProgressivePara/dataset.html).
See the website for license info.

It has been preprocessed by removing all meshes that contain a triangle with area smaller than machine epsilon (for double precision), that do not have disk topology, or contain topologically degenerate triangles.

Due to the size of the dataset, it is not included in the repository. Please download the dataset and paste it here to run the experiment.
