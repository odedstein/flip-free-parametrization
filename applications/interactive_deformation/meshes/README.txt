Attribution for the meshes contained in this folder:

The octopus 2D mesh was created using triangle [1] using the images in the Appendix of the thesis by Jacobson [2] using the MATLAB function mesh_from_png.m found in this folder.
The mouth mesh was created using triangle [1] from a mouth image from [publicdomainvectors.org](https://publicdomainvectors.org/en/free-clipart/Glossy-red-lips/80649.html) using the MATLAB function mesh_from_png.m found in this folder.
The robot mesh was created using triangle [1] from a robot image from [publicdomainvectors.org](https://publicdomainvectors.org/en/free-clipart/Blue-robot/62916.html) using the MATLAB function mesh_from_png.m found in this folder.
[1] Jonathan Shewchuck, 2005. "Triangle, A Two-Dimensional Quality Mesh Generator and Delaunay Triangulator." https://www.cs.cmu.edu/~quake/triangle.html
[2] Alec Jacobson, 2013. "Algorithms and Interfaces for Real-Time Deformation of 2D and 3D Shapes." PhD Thesis, ETH Zurich.

The hand tet mesh is by [Max Planck Society e.V via SMPL](https://smpl.is.tue.mpg.de/bodylicense), tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
The fish surface mesh "Blub" is a creation of [Keenan Crane](https://www.cs.cmu.edu/~kmcrane/Projects/ModelRepository/), and is in the public domain. It was tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
The cow surface mesh "Spot" is a creation of [Keenan Crane](https://www.cs.cmu.edu/~kmcrane/Projects/ModelRepository/), and is in the public domain. It was tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
