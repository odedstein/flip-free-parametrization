Attribution for the meshes contained in this folder:

The 2D meshes were created using triangle [1] using the images in the Appendix of the thesis by Jacobson [2] using the MATLAB function mesh_from_png.m found in this folder.
[1] Jonathan Shewchuck, 2005. "Triangle, A Two-Dimensional Quality Mesh Generator and Delaunay Triangulator." https://www.cs.cmu.edu/~quake/triangle.html
[2] Alec Jacobson, 2013. "Algorithms and Interfaces for Real-Time Deformation of 2D and 3D Shapes." PhD Thesis, ETH Zurich.

The octopus-vol tet mesh is by [Mark Pauly, "Shape Modeling with Point-Sampled Geometry"](https://dl.acm.org/doi/10.1145/882262.882319) via [libigl](https://github.com/libigl/libigl-tutorial-data), tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
The armadillo tet mesh is by the [Stanford 3D Scanning Repository](http://graphics.stanford.edu/data/3Dscanrep/), tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
The stuffedtoy tet mesh is by Stuffedtoy, by Ilya Baran via https://github.com/libigl/libigl-tutorial-data, tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
The hand tet mesh is by [Max Planck Society e.V via SMPL](https://smpl.is.tue.mpg.de/bodylicense), tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
The slug tet mesh is by [Indentional3D via Thingiverse](https://www.thingiverse.com/thing:1033690), tet-meshed with [TetWild](https://github.com/Yixin-Hu/TetWild).
